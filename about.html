About TaskTracker

TaskTracker is a simple yet powerful web application that helps you manage your daily tasks efficiently. You can add tasks, mark them as done, and delete them as needed.

This project is built with Python (Flask), HTML, CSS (Bootstrap), and SQLite for the backend database. It demonstrates user registration, login, session management, and CRUD operations.

TaskTracker is developed as part of the CS50x Final Project, with the goal of solving real-life productivity problems.
